# Services de Maintenance SGS — site Jekyll

Template Jekyll en français basé sur le concept sombre orange.

## Installation

```bash
bundle install
bundle exec jekyll serve
```

Puis ouvrez http://localhost:4000

## Modifier les infos

- `_config.yml` : nom, téléphone, courriel, ville
- `index.md` : contenu de la page d’accueil
- `assets/css/style.css` : couleurs et mise en page
- `assets/images/` : images SVG de remplacement

## Déploiement

Compatible avec GitHub Pages si vous utilisez une action Jekyll, Netlify, Cloudflare Pages ou tout hébergement statique.
